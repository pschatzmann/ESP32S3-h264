using H264DecoderPSRAM = H264Decoder<PSRAMAllocator<uint8_t> >

Type alias for H264Decoder using default PSRAM allocation.

Provides a simpler way to declare H264Decoder instances without specifying the allocator template parameter. Uses PSRAMAllocator<uint8_t> by default.

Example usage:

H264DecoderPSRAM decoder(&udp); // Instead of H264Decoder<PSRAMAllocator<uint8_t>>
