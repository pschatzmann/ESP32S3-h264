Detailed Description

RTPPacketizer class to packetize H.264 NAL units into RTP packets. This class takes raw H.264 NAL units (in Annex-B format) and packetizes them into RTP packets suitable for streaming over UDP.

This way you can e.g. Send H264 frames to QGroundcontrol
